# Grid Layout Capacity

Shortcut Page capacity is defined by the user's selected Grid Layout rather than inferred from measured viewport capacity. The Grid Layout records preset/custom mode plus numeric rows and columns, so page size is predictable to users while the tile presentation can scale down to preserve the no-browser-scroll New Tab Surface constraint.

The default Grid Layout is the 2x6 preset. Column spacing, line spacing, and icon size are independent percentage-based presentation settings that apply to both preset and custom Grid Layouts.
